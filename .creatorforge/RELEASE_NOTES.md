# Release Notes

## v0.6 Alpha

CreatorForge v0.6 Alpha delivers a local-first YouTube content studio powered by Ollama.

### Major Capabilities

- Topic-to-project generation with selectable local models.
- Research, research summary, outline, titles, script, description, tags, and thumbnail prompt artifacts.
- Context-aware project generation using distilled research.
- Project metadata, local saving, history, reopening, and legacy-project fallback.
- Live stage-level generation progress.
- Output-focused prompt quality controls and cleanup.

### Scope

This Alpha focuses on local creator workflow quality. Publishing, analytics, collaboration, and multi-platform workflows remain planned work.
