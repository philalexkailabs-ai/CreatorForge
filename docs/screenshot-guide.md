# Screenshot Guide

## Dashboard — `assets/screenshots/dashboard.png`

Show the full Studio layout: sidebar, topic input, model selector, generation controls, and several populated artifact cards.

## Generation — `assets/screenshots/generation.png`

Show an active project generation with visible loading artifacts and the progress card.

## Progress — `assets/screenshots/progress.png`

Focus on the Generation Progress card with completed, current, and waiting stages visible.

## History — `assets/screenshots/history.png`

Show the Saved Projects sidebar containing project names and a reopened project displayed in existing cards.

## Demo GIF — `assets/screenshots/demo.gif`

Use the sequence described in [Demo Guide](demo-guide.md).

## Capture Rules

- Use a realistic but non-sensitive topic.
- Avoid personal paths, credentials, or private generated content.
- Keep browser chrome out of the image unless it adds useful context.
- Use consistent dark-mode styling and readable zoom.
