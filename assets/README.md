# CreatorForge Assets

This folder stores repository presentation assets only. It does not contain application runtime assets.

- `logos/` — CreatorForge marks for light and dark backgrounds.
- `screenshots/` — README, documentation, and release-demo screenshots or GIFs.
- `branding/` — banners, social preview images, palettes, and source-ready branding references.

See [GitHub Branding Guide](../docs/github-branding.md) and [Screenshot Guide](../docs/screenshot-guide.md).
