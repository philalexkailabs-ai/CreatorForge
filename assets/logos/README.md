# Logo Assets

Add approved CreatorForge logo files here:

- `logo-light.png` — dark logo for light backgrounds.
- `logo-dark.png` — light logo for dark backgrounds.
- `logo-mark.png` — square icon or favicon source.
- `logo.svg` — optional scalable source when the logo system is approved.

Keep transparent backgrounds where possible. Do not add unapproved logo variants to the README.
